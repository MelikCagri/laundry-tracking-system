# Proje Başlatma Yönergesi (Initial Setup Directive)

## Amaç
`spec.md` dosyasında belirtilen KYK Çamaşırhane Takip Sistemi'nin temel yapısını kurmak.

## Girdi
- `spec.md` (Teknik Şartname)

## Adımlar
1. **Frontend Kurulumu:** React (Vite) + TypeScript + Tailwind CSS kurulumu.
2. **Backend Kurulumu:** Node.js + Express.js temel yapısının kurulması.
3. **Veritabanı Hazırlığı:** PostgreSQL şemalarının planlanması.

## Kullanılacak Araçlar (Execution Scripts)
- Henüz bir execution script'i tanımlanmadı. İhtiyaç halinde `execution/` altında oluşturulacak.
